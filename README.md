# Thyroid Disease Detection using Machine Learning

## About the Project

This project focuses on detecting thyroid conditions (benign or malignant) using ultrasound images. The goal was to build a simple machine learning model that can take an image as input and give a prediction.

I worked on this as part of my learning in machine learning and image processing, keeping the implementation simple and easy to understand.

---

## What it does

* Takes an ultrasound image as input
* Processes the image using basic techniques
* Uses a trained model to classify the condition
* Displays whether the result is benign or malignant

---

## Tech Stack

* Python
* OpenCV
* NumPy
* Scikit-learn / TensorFlow
